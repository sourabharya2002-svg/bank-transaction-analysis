# Project Explanation - Bank Transaction Analysis

## Introduction
This project focuses on analyzing bank transaction data using SQL and Excel.

## Objective
The objective is to identify transaction patterns, customer behavior, and high-value transactions.

## Dataset
The dataset includes transaction ID, account ID, transaction type, amount, city, and transaction date.

## SQL Analysis
- Total Transactions & Amount
- Credit vs Debit Analysis
- City-wise Analysis
- Account Balance Calculation using CASE
- Monthly Trend Analysis
- High Value Transactions
- Highest Transaction by City
- Top 3 Transactions per City using DENSE_RANK

## Tools Used
- MySQL
- Excel
- PowerPoint

## Conclusion
This project helps in understanding financial patterns and demonstrates how SQL and Excel can be used for data analysis.
